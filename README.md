# Google_Search_Analysis-pytrends-used-

Built a real-time Google Search Trend Analysis tool using the Pytrends API — an unofficial Google Trends API wrapper — to extract, 
analyze, and visualize global search interest data for any keyword.
The project simulates a data-driven job role scenario where search behavior is analyzed across countries, time periods, and 
competing keywords to support strategic business and marketing decisions.

Cloud Computing shows consistent global search interest with identifiable seasonal peaks
Data Science and Machine Learning follow similar trend patterns, indicating overlapping audience interest
Geographic analysis reveals concentrated search demand in specific regions, useful for targeted digital marketing strategies
